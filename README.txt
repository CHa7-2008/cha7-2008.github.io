
CHa7 / 茶柒 — Apple 风个人主页（简约 + 沉浸式音乐展示）

说明
----
这是为你定制的静态个人主页模板（Apple 风格），可直接部署到 GitHub Pages。

包含文件：
- index.html
- assets/style.css
- assets/script.js
- assets/qr.png  （占位二维码图片，请替换为你的微信二维码）
- README.txt

快速上手（推荐流程）
1. 下载并解压此压缩包。
2. 打开文件夹，替换以下占位内容：
   - assets/qr.png -> 替换为你的微信二维码图片（png/jpg）
   - 在 index.html 中替换 hero-art 区块或将自己的图片命名为 assets/hero.jpg 并修改 html
   - 在音乐卡片的 <a class="listen"> 链接中填入你每首歌在网易云的 song id 或 album id（PC 版链接）
     示例 PC 单曲链接格式: https://music.163.com/#/song?id=SONG_ID
     示例 专辑链接格式: https://music.163.com/#/album?id=ALBUM_ID

3. 将整个文件夹上传到 GitHub 仓库 `cha7.github.io`（在仓库页面选择 "Add file" -> "Upload files"）。
4. 上传完成后，进入仓库 Settings -> Pages（或直接访问 https://cha7.github.io），Github Pages 会自动发布。

提示与替换建议
- 我在模板中使用了你提供的短链作为 Douyin 快捷跳转（会直接跳转短链）。若你需要我帮忙把短链扩展为主页长链，我可以在帮你确认后修改。
- 字体采用系统级别 Apple 风字体栈，并用粗体大标题模拟 iPhone 宣传图的大粗体。
- 若要我替你把网易云每首歌曲的具体跳转链接填上（更精确），请把对应单曲在网易云的“song id”或完整 PC 链接发给我，我会帮你替换。

如需我代为上传至仓库并完成发布（我会告诉你具体每一步的点击位置），我可以一步步引导你操作。
